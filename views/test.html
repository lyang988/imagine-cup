Here are two functions that print the bigger of the two arguments:

Function 1:

def print_bigger(a, b):
if a > b:
 print(a)
else:
 print(b)

------

Function 2:

java.io.*;
void printBigger(int a, int b){
 if (a > b) {
  System.out.println(a); }
 else {
  System.out.println(b); }
}

Notice that the overall structure of the code is very similar, but the specific syntax is quite different. For instance, instead of using colons and indentation, Java uses curly brackets (‘{‘, ‘}’) to specify function and if-statement bodies. Also, statements in Java end in a semicolon (‘;’). There are some other big differences that we will explore soon, but these two are the most visible - and easy to forget!
